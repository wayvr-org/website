## Slider component

### `<Slider>`

### A simple slider.

#### Parameters

`min_value`: **float**

`max_value`: **float**

_Needs to be bigger than `min_value`_

`value`: **float**

_Initial primary slider value_

`value2`: **float**

_Initial secondary slider value (for range sliders)_

`show_value`: "1" | "0"

_Set to 0 if you don't want to display value text in the slider handle_
