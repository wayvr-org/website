# Configuration

The sample configuration file is located at [wayvr/src/res/config.yaml](https://github.com/wlx-team/wayvr/blob/main/wayvr/src/res/config.yaml).

Place this file into `~/.config/wayvr/conf.d` in order to make changes:

```sh
wget -O ~/.config/wayvr/conf.d/base.yaml https://raw.githubusercontent.com/wlx-team/wayvr/refs/heads/main/wayvr/src/res/config.yaml
```

The file is self-documenting, opening and reading it will hopefully answer your questions.

For further questions, reach out to us on Discord or Matrix.
